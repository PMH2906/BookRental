CREATE DATABASE library;

SELECT database();

USE library;

CREATE TABLE `book` (
`rank` SMALLINT(20) UNSIGNED,
`book_id` BIGINT(20),
`title` VARCHAR(20),
`author` VARCHAR(20),
`field` VARCHAR(20),
`rental_status` BOOLEAN -- BOOLEAN 및 BOOL은 동의어이기 때문에 TINYINT (1)
);

속성으로 UNSIGNED은 칼럼내에서 음수를 포함하지 않거나 혹은 수의 range를 양수쪽으로 더 넓게 가지고싶을때 사용한다.
예시) 엔드포인트가 -2147483648 부터 2147483647 에서 --> 0 부터 4294967295. 로 늘어나게된다.
